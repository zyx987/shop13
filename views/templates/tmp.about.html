    This is an e-shop application for guitar and related stuff:

  • The menu is generated according to database entries -> a request to 'api/categories' will return the menu tree with categories and their corresponding subcategories (menu is two steps deep). So if the shopkeeper / database maintenance clean up the database, unused (sub)categories won't be displayed anymore automatically
  • JQUERY denied :) This application does not use Jquery (only jqlite from angularjs) just load libraries you really need!
  • Cool / easy checkout with no unnecessary stuff
  • Shopping-cart with all info's including shipping
  • Responsive design for desktop / mobile
  • Quick search function
